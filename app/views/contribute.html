Pre-requisites: Install dependencies

Java 1.6 (older Java versions do not work).

Git 1.7 (other Git versions work as well).

Maven 3 (Maven 2 does not work).

Check our REST API and documentation

Build your own application based on DMX
  1. Download DMX distribution from: https://www.digitalmemex.com/dmx-0.0.2-SNAPSHOT.tar.gz
  2. Extract the archive
  3. Start the server in foreground with `bin/karaf` or as a service with `bin/start`(hit <CTRL>+<D> or call `bin/stop` to stop it).
  4. Open http://localhost:17071/de.deepamehta.webclient

    To login fill username: admin and leave the password blank

Then

  1. Create your repository by creating a topic Repository with the name of your appplication and the url of your code
  2. Open the repository manager at http://localhost:17071/dmx and install your app

Access to your code

cd dmx-0.0.2-SNAPSHOT/deepamehta-files/dmx

Now you can continue building your own application!!!!

If you develop an add-on we would be happy if you tell us, by sending a mail to teamATdigitalmemex.com

Contribute with the open tasks of DMX distribution

If you found something you want to work on or to fix, fork the repository and submit a pull request with your changes

  1. Check out the issues
  2. Fork the code from https://github.com/digitalmemex/distribution
  3. Do a pull request

We have have a beginner-friendly tag for things that we believe are simple enough to get started.

Git Workflow

To contribute make sure you follow this workflow, thanks!!

  1. Have a master branch
  2. Have a develop branch
  3. Each small feature is done in a new branch (put the issue number as part of the branch name)
  4. Do a Pull Request over each small feature
